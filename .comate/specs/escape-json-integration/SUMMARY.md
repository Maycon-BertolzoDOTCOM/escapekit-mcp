## 🚀 Usage Examples

### CLI Usage

```bash
# Generate with escape.json (default)
escapekit generate analysis.json

# Generate without escape.json
escapekit generate analysis.json --no-escape-json

# Generate with custom output
escapekit generate --code src/app.ts --output ./my-project --escape-json
```

### Programmatic Usage

```typescript
import { ProjectGenerator } from './generators/ProjectGenerator.js';

const generator = new ProjectGenerator();
await generator.generate({
  rootPath: './output',
  projectName: 'my-project',
  analysisResult,  // Required for escape.json
  resolutions,
  transformations,
  kiwiTestRunId: 8,
  generateEscapeJson: true,  // Default
});
```

## 🎓 Lessons Learned

### Technical

1. **TypeScript Type Mismatch**
   - `AnalysisResult.issues` is an array, not object-by-file
   - Required fixing `convertIssues()` and `calculateIssueBreakdown()`
   - Lesson: Always verify data structure assumptions

2. **Test Isolation**
   - Need to clean up between tests (`beforeEach` not `beforeAll`)
   - Prevents state leakage
   - Lesson: Each test should be independent

### Process

1. **Incremental Integration**
   - Started with basic structure
   - Added tests early
   - Fixed bugs iteratively
   - Lesson: Test-driven integration works well

2. **Documentation First**
   - Protocol documented before implementation
   - Schema defined clearly
   - Lesson: Clear specs prevent rework
