# 📚 Knowledge Base - EscapeKit

## Propósito
Este diretório armazena conhecimento estratégico, research e documentação técnica para orientar decisões de desenvolvimento futuras.

## Arquivos

### `chinese-ai-research.md`
- **Arsenal estratégico** de papers e técnicas chinesas
- **Gatilhos baseados em métricas** para implementação
- **Tempo estimado** por melhoria
- **Regra de ouro**: "Só implementar se métrica real indicar necessidade"

## Filosofia de Uso

### ✅ Uso Correto
- Consultar quando métricas reais indicarem necessidade
- Usar como referência para otimizações específicas  
- Manter atualizado com novas pesquisas relevantes

### ❌ Uso Incorreto
- Implementar features "porque o paper é legal"
- Parar desenvolvimento para estudar tudo de uma vez
- Adicionar complexidade sem necessidade demonstrada

## Processo de Decisão

1. **Medir métrica atual** (acurácia, fidelidade, custo, latência)
2. **Comparar com baseline** definida
3. **Se fora do esperado** → Consultar knowledge base
4. **Escolher solução mais adequada** do arsenal
5. **Implementar incrementalmente**
