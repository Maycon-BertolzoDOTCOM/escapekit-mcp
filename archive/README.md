# archive/

Este diretório contém material histórico preservado que não faz parte do fluxo ativo de desenvolvimento.

## Critério de arquivamento

Um artefato é movido para `archive/` quando:
- É um pacote cujo escopo ou produto foi renomeado/substituído (ex: `@codememoria/` → `@escapekit/`)
- É uma documentação operacional obsoleta que não reflete o estado atual do código
- É um subprojeto descontinuado que precisa ser preservado para referência histórica

## Conteúdo

- `packages/` — Pacotes de escopos descontinuados (ex: `@codememoria/`)
