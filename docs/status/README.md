# docs/status/

Este diretório contém relatórios operacionais, status de projetos e documentos não-permanentes.

Diferente dos guias em `docs/`, os documentos aqui podem ficar desatualizados — eles registram o estado em um momento específico.

## Conteúdo

Documentos de status são movidos para aqui quando são operacionais/temporários em vez de referenciais permanentes.
